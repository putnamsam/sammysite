# sammysite.com starter

This project is a lightweight static site for `sammysite.com`.

## Files

- `index.html` contains the page structure and content.
- `styles.css` contains the full visual design and responsive layout.
- `script.js` adds reveal-on-scroll motion.

## Deploy

You can deploy this site to any static host, including:

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

If your host supports custom domains, point `sammysite.com` at the deployed site in that provider's domain settings and DNS panel.

## Customize

- Replace the placeholder copy in `index.html`.
- Update `hello@sammysite.com` to your real email.
- Add real project links, images, or sections as needed.
